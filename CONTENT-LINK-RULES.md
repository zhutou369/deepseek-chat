# deepseek-chat.com 内链规则

## 支柱教程（优先链向）

| 路径 | 适用场景 |
|------|----------|
| `/posts/deepseek-chat-web-basics/` | 首条消息、网页入门、模型选择 |
| `/posts/deepseek-multi-turn-memory/` | 多轮、上下文、新开对话 |
| `/posts/deepseek-mobile-app-chat/` | 手机、App、弱网、语音 |
| `/posts/deepseek-chat-scenario-library/` | 模板、Prompt、场景 |
| `/posts/deepseek-chat-export-share/` | 导出、分享、隐私、脱敏 |

## 每篇要求

- 站内内链 2～5 条，锚文本自然
- 外链仅官方 `chat.deepseek.com`、`platform.deepseek.com`
- 禁止链到其他 deepseek-*.com 站群
- 文末可加「## 延伸阅读」链到相关支柱

## 图片

- 封面：`coverImage: "/static/posts/xxx-cover.svg"`
- 正文步骤图：`/static/posts/covers/step-guide.svg` 或主题 step 图
- 禁止正文使用 unsplash.com 外链
