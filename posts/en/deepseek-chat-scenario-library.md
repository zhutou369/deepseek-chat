This site focuses on **chat scenarios**, not API integration. Paste the templates below into DeepSeek web chat and replace bracketed placeholders.

## Writing & content

**Hook rewrite for social posts**

```
You are a senior editor. Rewrite the draft below into a compelling opening, under 150 words. Keep facts; no hype.
---
[paste draft]
```

**Meeting notes extraction**

```
Below is a meeting transcript. Output:
1. Decisions (table: item / owner / deadline)
2. Open questions (max 5)
---
[paste transcript]
```

## Coding & debugging

**Explain an error**

```
I am a junior Python developer. Given the error and code below:
1. Explain the cause in plain language
2. Suggest the smallest fix
3. Do not rewrite the whole file
---
Error: [paste]
Code: [paste]
```

**Code review**

```
Review the function below on security, performance, and readability — max 2 notes per area:
[paste code]
```

## Learning & exams

**Reading notes**

```
I finished chapter [N] of [book title]. Provide:
- 3 core ideas
- 1 actionable takeaway
- 1 point I might misunderstand
Total under 300 words.
```

**Interview practice**

```
You are the interviewer for [role]. Ask one question at a time; after my answer, brief feedback, then next question. Start now.
```

## How to use this library

1. Copy the closest template
2. Change only brackets and attachments
3. If round one is weak, iterate with [multi-turn tips](/en/posts/deepseek-multi-turn-memory/)

For API keys, rate limits, and developer setup, use dedicated API documentation; this site covers everyday web chat.
