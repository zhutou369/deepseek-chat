问「帮我制定学习计划」会得到鸡汤。换成「每天 1 小时、共 7 天、目标 Python 读写 CSV」才可执行。

## 周计划模板

```

目标：7 天内学会用 Python 读写 CSV 并做简单统计。

我每天可投入 1 小时，零基础。

请输出 Markdown 表格：日期/学习内容/练习/验收标准。

不要安排超过 1 小时的任务。

```
