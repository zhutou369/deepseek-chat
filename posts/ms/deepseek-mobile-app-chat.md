Sembang mudah alih sering bermaksud "edit salinan dalam perjalanan". Skrin kecil memerlukan prompt **lebih pendek dan berstruktur**.

## Pintu masuk mudah alih

1. **Pelayar mudah alih**: Chrome/Safari → `chat.deepseek.com`
2. **Pelayar dalam aplikasi**: log masuk mungkin tidak dikongsi dengan pelayar sistem
3. **App rasmi** (jika ada): biasanya lebih stabil

## Prompt sesuai telefon

- Satu subtugas setiap mesej
- Gunakan senarai bernombor
- Minta "3 poin utama" dahulu, kemudian kembangkan

## Input suara

- Semak kesilapan transkripsi
- Istilah teknikal: eja dengan jelas
- Suara untuk idea; taip untuk kekangan akhir

## Rangkaian lemah

- Hantar mesej panjang hanya apabila stabil
- Jika terputus: "Sambung dari poin 3; jangan ulang teks sebelumnya"
- Salin hasil penting ke nota (lihat [eksport](/ms/posts/deepseek-chat-export-share/))

## Masalah biasa

| Masalah | Tindakan |
|---------|----------|
| Halaman kosong | Tukar pelayar atau kosongkan cache |
| Papan kekunci menutup input | Landscape atau pendekkan input |
| Jawapan terpotong | "Output dua bahagian; bahagian 1 dahulu" |
