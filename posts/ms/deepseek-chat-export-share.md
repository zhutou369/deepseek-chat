Sembang sering mengandungi data pelanggan atau salinan dalaman. **Semak redaksi sebelum eksport atau kongsi.**

## Kaedah sandaran

1. **Salin manual** — sesuai untuk satu jawapan
2. **Ringkasan Markdown** — minta model ringkaskan perbualan panjang
3. **Tangkapan skrin** — semak maklumat akaun di sisi

## Senarai semak sebelum kongsi

- [ ] Buang telefon, e-mel, ID kerajaan
- [ ] Ganti nama syarikat dengan "Syarikat A"
- [ ] Buang API key, kata laluan, URL dalaman
- [ ] Sahkan khalayak (awam vs pasukan)

## Privasi pelayar

- Log keluar pada PC awam; kosongkan cookie `chat.deepseek.com`
- Jangan tampal rahsia produksi dalam sembang
- Rangkaian korporat mungkin memantau trafik — kendalikan data sensitif secara tempatan

## Kosongkan jejak tempatan

1. Tetapan pelayar → privasi → kosongkan data laman
2. Mudah alih: tutup tab; kosongkan cache WebView jika perlu
3. Semak penyegerakan papan keratan awan
