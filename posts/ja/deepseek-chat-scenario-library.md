本サイトは **チャットシナリオ** に特化（API 連携ではありません）。以下を DeepSeek ウェブチャットに貼り、[] 内を置き換えてください。

## 執筆・コンテンツ

**導入文のリライト**

```
あなたは経験豊富な編集者。下書きを150字以内の魅力的な導入に。事実は維持、誇張禁止。
---
[下書き]
```

**議事録抽出**

```
会議文字起こしから：
1. 決定事項（表：項目/担当/期限）
2. 未決事項（最大5）
---
[文字起こし]
```

## コーディング

**エラー説明**

```
Python 初心者です。エラーとコードから：
1. 原因を平易に
2. 最小修正案
3. ファイル全体は書き直さない
```

## 学習

**読書メモ**

```
『[書名]』第[N]章読了。核心3点、実行可能な提案1つ、誤解しやすい点1つ。合計300字以内。
```

**面接練習**

```
[職種]の面接官として。1問ずつ、回答後に短いフィードバック。
```

## 使い方

1. 近いテンプレをコピー
2. [] と添付のみ変更
3. 不満なら [マルチターンのコツ](/ja/posts/deepseek-multi-turn-memory/) で反復

API キー等は別ドキュメントを参照。本サイトは日常チャット向けです。
